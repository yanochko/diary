# 🗓️ Ежедневник (Daily Planner)

## 📌 Описание
Простое веб-приложение "Ежедневник", позволяющее добавлять, просматривать и удалять задачи.  
Интерфейс реализован в виде **статической HTML-страницы** с использованием:
- **HTML5** — семантическая разметка
- **CSS3** — адаптивная верстка
- **JavaScript (Vanilla)** — базовое взаимодействие

---

## ✅ Функциональность

- ✅ Добавление задач с названием (обязательно) и описанием (необязательно)
- ✅ Удаление задач из списка
- ✅ Адаптивный дизайн (от 320px до 1920px)
- ✅ Карточки задач с кнопками "Удалить"
- ✅ Без перезагрузки страницы (используется `preventDefault`)

---

## 📱 Адаптивность

- Мобильные устройства: от **320px**
- Планшеты: до **768px**
- Десктопы: до **1920px**

---

## 🧪 Тестирование

1. Открой `index.html` в браузере
2. Введи название задачи → нажми "Добавить"
3. Задача появится в списке
4. Нажми "Удалить" — задача исчезнет
